h2. Como ver la presentación

Puedes acceder directamente a la presentación en pdf "*aqui*":http://github.com/pacoguzman/ruby_testing

O puedes verla en tu navegador utilizando "*showoff*":http://github.com/schacon/showoff

* gem install showoff
* git clone git://github.com/pacoguzman/ruby_testing.git
* cd ruby_testing
* showoff server
* Y abrir en tu navegador "*http://localhost:9090*":http://localhost:9090
